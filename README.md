# bdsm_extension

Extension to do BD tests on LMS

## 🛠 Ручне встановлення (на постійній основі)

1. Відкрийте браузер (Chrome, Brave, Edge).
2. Перейдіть за адресою `chrome://extensions/`.
3. Увімкніть **Developer mode** (режим розробника) у правому верхньому куті.
4. Натисніть **Load unpacked** (Завантажити розпаковане розширення).
5. Виберіть папку з цим проектом.

Тепер розширення буде доступне у вашому браузері постійно.

## Можливості

- Автоматично заповнює відповіді з `answers.json`.
- **Режим повільного заповнення (Slow Filling)**: заповнює одне питання кожні 10 секунд для імітації поведінки людини. Можна увімкнути/вимкнути через меню розширення.
- **Миттєве заповнення (Foce fill):** якщо набридло чекати 10 секунд і ви вимкнули цей режим, то в тому ж меню розширення можете натиснути кнопку, яка одразу дасть відповідь на усі питання.
